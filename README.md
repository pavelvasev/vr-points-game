# VR Points game

## Control
* click with mouse
* move mouse while Ctrl key is pressed
* click with XR controllers

## Open in VR mode
* click "Enter VR" semi-transparent button 
in the middle bottom of the screen.

## Run online

https://viewlang.ru/viewzavr-apps/vr-points-game

## Run on own server

* clone
* start-http-server.sh
* open `index.html`
